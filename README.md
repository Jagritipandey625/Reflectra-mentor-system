# Reflectra-mentor-system

Reflectra is a **mentor-style web application** designed to help users **reflect on their daily, weekly, and monthly plans**, track alignment with actual activities, and receive **non-judgmental, adaptive feedback**. It focuses on behavioral patterns, energy impact, and intention vs execution, supporting **reflection, disagreement handling, and voice-based interactions**.

---

## Features

- **Daily, Weekly, Monthly Planning**: Track your priorities without overwhelm.
- **Mentor Evaluation Engine**: Compares planned intentions vs actual activities and provides insightful feedback.
- **Adaptive Reflection**: Users can disagree with feedback; system adapts based on user input.
- **Voice Input & Output**: Speak your plan or listen to mentor feedback using Web Speech API.
- **Behavioral Insights**: Detects patterns in productivity, energy, and distraction over time.
- **Accessible and Human-Centric Design**: Non-judgmental guidance for sustainable productivity.

---

## How It Works

1. User enters **daily plan** and logs activities with time spent and energy impact.
2. System compares **intentions vs reality**, calculates focus and energy scores.
3. Feedback is generated in **mentor style**, explaining patterns and offering advice.
4. Users can **agree or disagree**; disagreement triggers reflection and improves future evaluation.
5. Weekly and monthly summaries highlight patterns and long-term insights.
